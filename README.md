# ADAComics 👩‍💻🚀
5to proyecto: una aplicación que permita listar cómics y personajes de comics, y obtener información sobre los mismos a través de una API REST con datos reales.

# Tecnologías 💾🤖
HTML, Css, Sass y Git.
